SF-Plugin WebUI (本地版)

这是一个基于云崽的智能对话应用，支持本地部署访问。

功能特点：
  • 支持文本对话 (SS/GG/DD模式)
  • 支持图片识别和上传
  • 支持Markdown格式渲染
  • 支持数学公式 (LaTeX)
  • 本地部署，无需代理
  • 支持IPv6直连
使用说明：
  1. 输入您的QQ号（向 Bot 发送 #sf登录 获取验证码）
  2. 如服务端设置了访问密码，请输入密码
  3. 点击连接按钮
  4. 选择对话模式(SS/GG/DD)
  5. 开始对话！
